# 🎬 CinemaCityHub

A powerful Telegram Movie + Anime Filter Bot built with Python, MongoDB Atlas, and python-telegram-bot.

## ✨ Features

* 🎬 Movie & Anime Search
* 🌐 Language Filters

  *Tamil
  * Hindi
  * English
  * Telugu
  * Malayalam
  * Kannada
  * Dual Audio
  * Multi Audio
* 📽 Quality Filters

  * 4K
  * 1080p
  * 720p
  * 480p
  * 360p
* ⭐ IMDb Integration

  * Poster
  * Rating
  * Year
  * Genre
  * Plot
* 📄 Pagination System
* 🔒 Force Subscribe
* ⏳ Auto Delete
* 📩 Movie Request System
* 📊 Admin Panel
* 📢 Broadcast System
* 🚫 Global Ban System
* 🛡 Anti Spam Protection
* ☁️ Render Deployment Ready

## 🛠 Environment Variables

* BOT_TOKEN=
* MONGO_URI=
* DB_CHANNEL=
* FORCE_SUB_CHANNEL=
* UPDATES_CHANNEL=
* SUPPORT_GROUP=
* ADMIN_ID=
* OMDB_API_KEY=


## 🚀 Deployment

1. Upload source code to GitHub
2. Connect repository to Render
3. Add Environment Variables
4. Deploy
